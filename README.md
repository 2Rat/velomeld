# Velomeld - Radwegecheck fuer Buerger

Probleme, Wuensche und positive Erfahrungen zum Radverkehr melden - direkt vom Smartphone.

## Features
- GPS-genaue Standorterfassung
- Foto-Dokumentation
- Offline-faehig
- Export als GeoJSON, CSV, QGIS
